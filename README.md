

## Getting Started

 1. Install the library:
    
    ```bash
    
    npm  install
    
    ```
 2. Rename 
    
    > next-env.d.example
    
    to
    
    > next-env.d.ts
    
 3. Rename
    
    > .env.example
    
    to

    >.env

	and fill the 'NEXT_PUBLIC_API' with your API url
 
 4.  Run the development server:
		```bash

		npm  run  dev

		# or

		yarn  dev

		# or

		pnpm  dev

		```
 5. Open [http://localhost:3002](http://localhost:3002) with your browser to see the result.
